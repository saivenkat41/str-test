String Programming Challenge

Test your string manipulation skills with these coding problems

Easy
1. Reverse a String
Write a function that takes a string as input and returns the string reversed.

Example: "hello" → "olleh"
Think about how you can iterate through the string from the end to the beginning. In JavaScript, strings are immutable, so you might need to convert to an array first.
Easy
2. Palindrome Check
Write a function that checks if a given string is a palindrome (reads the same backward as forward).

Example: "madam" → true, "hello" → false
Compare characters from the start and end moving towards the center. Remember to ignore case and non-alphanumeric characters if needed.
Easy
3. Count Vowels
Write a function that counts the number of vowels in a given string.

Example: "apple" → 2, "why" → 0
Consider both uppercase and lowercase vowels. You can create a set of vowels to check against.
Easy
4. Capitalize Words
Write a function that capitalizes the first letter of each word in a string.

Example: "hello world" → "Hello World"
Split the string into words, capitalize the first letter of each, then join them back. Or iterate through the string and capitalize letters that follow a space.
Easy
5. Find Substring
Write a function that checks if a substring exists in a string and returns its starting index, or -1 if not found.

Example: "hello", "ll" → 2
You can use built-in string methods, or implement a simple sliding window approach.
Medium
1. Longest Substring Without Repeating Characters
Given a string, find the length of the longest substring without repeating characters.

Example: "abcabcbb" → 3 ("abc"), "bbbbb" → 1 ("b")
Use a sliding window approach with a hash map to track character positions. Adjust the window when a duplicate is found.
Medium
2. String to Integer (atoi)
Implement the atoi function which converts a string to an integer. Handle leading whitespace, optional +/- sign, and stop at first non-digit.

Example: "42" → 42, " -42" → -42, "4193 with words" → 4193
Process the string character by character, building the number digit by digit. Don't forget to handle overflow cases.
Medium
3. Group Anagrams
Given an array of strings, group anagrams together.

Example: ["eat", "tea", "tan", "ate", "nat", "bat"] → [["eat","tea","ate"],["tan","nat"],["bat"]]
Anagrams have the same character counts. Consider using sorted strings as keys in a hash map.
Medium
4. Longest Palindromic Substring
Given a string, find the longest substring which is a palindrome.

Example: "babad" → "bab" or "aba", "cbbd" → "bb"
Consider expanding around each character (and between characters) to find palindromes. There's a O(n²) solution with O(1) space.
Medium
5. String Compression
Compress a string by replacing consecutive duplicate characters with the character followed by count. If compressed string is not smaller, return original.

Example: "aabcccccaaa" → "a2b1c5a3", "abc" → "abc"
Iterate through the string while counting consecutive characters. Build the compressed string as you go, and compare lengths at the end.
Hard
1. Regular Expression Matching
Implement regular expression matching with support for '.' and '*'. '.' matches any single character. '*' matches zero or more of the preceding element.

Example: "aa", "a" → false; "aa", "a*" → true; "ab", ".*" → true
This problem requires dynamic programming or recursion with memoization. Consider all cases when encountering a '*'.
Hard
2. Minimum Window Substring
Given two strings S and T, find the minimum window in S which contains all characters of T.

Example: S = "ADOBECODEBANC", T = "ABC" → "BANC"
Use a sliding window approach with two pointers. Track character counts with hash maps. Expand the window when missing characters, contract when all are found.
Hard
3. Word Break
Given a string and a dictionary of words, determine if the string can be segmented into space-separated dictionary words.

Example: "leetcode", ["leet", "code"] → true
Dynamic programming works well here. Track which positions in the string are break points. For each position, check all possible previous break points.
Hard
4. Longest Valid Parentheses
Given a string containing just '(' and ')', find the length of the longest valid (well-formed) parentheses substring.

Example: "(()" → 2, ")()())" → 4
Use a stack to track indices of unmatched parentheses, or use dynamic programming. The stack approach is more straightforward.
Hard
5. Edit Distance
Given two strings, find the minimum number of operations (insert, delete, replace) to convert one string to another.

Example: "horse", "ros" → 3 (horse → rorse → rose → ros)
This is a classic dynamic programming problem. Create a 2D DP table where dp[i][j] represents the edit distance between the first i characters of word1 and first j characters of word2.
